Frequently asked questions?

What is the difference between HTML, CSS, and JavaScript?

HTML is for structure, CSS is for styling, and JavaScript is for behavior interactivity.

What is responsive design?

An approach to web design that makes web pages render well on a variety of devices and window or screen sizes.

Explain the concept of the Document Object Model (DOM)?

The DOM is an interface that browsers implement to represent the structure of a document (like HTML) as a tree of objects, allowing scripts to manipulate content, structure, and styles.

What is the CSS Box Model?

It describes the rectangular boxes generated for elements in the document tree, consisting of margins, borders, padding, and the actual content.

What is a single-page application (SPA)?

A web app that loads a single HTML page and dynamically updates content as the user interacts with the app, often using frameworks like React, Vue, or Angular.

What are media queries?

Media queries use the @media rule followed by conditions (media features) and CSS styles.

What is the difference between front-end and back-end development?

1. Front-end development focuses on the visual elements and user experience of a website. It involves working with HTML, CSS, and JavaScript.
2. Back-end development deals with the server-side logic, databases, and application architecture. It often involves languages like PHP, Python, Ruby, or Java.
